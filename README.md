# Gold Price Forecasting Using Historical Time Series Data
## Business Problem
Gold prices fluctuate rapidly, making short-term price movements difficult to predict.
Traders and analysts need data-driven forecasts to support timely and informed decisions.
This project demonstrates how historical price data can be used to forecast future gold price trends.
Data & Method
•	Historical daily gold closing price data (USD)
•	Time series forecasting approach to learn price patterns and trends
•	Model trained on past data to generate short-term future price forecasts
An advanced time series model was used to capture trends and momentum in gold prices.
Results
•	The model successfully captures historical gold price trends
•	Predicted prices closely follow actual price movements
•	A short-term future price forecast was generated based on learned patterns
Key Insights
•	Gold prices show strong dependence on recent historical momentum
•	The forecast highlights potential short-term upward and downward movements
•	These predictions can support better timing and risk-aware decision-making
Deliverables & Insights

1.	Forecast Chart – Last 60 days historical gold prices + 15-day predicted prices.
2.	CSV File – Daily predicted prices for the next 15 days.
3.	Weekly Summary – Average, min, max prices, and trend analysis for each week.
4.	Text Summary – Key insights and expected trends for actionable decisions.
